# 🌐 Tor Web Platform 
### Next-Generation Web Development Solutions

ئەم پڕۆژەیە وێب سایتێکی مۆدێرنە بۆ نمایشکردنی قالبەکان و خزمەتگوزارییەکانی دروستکردنی وێب سایت.



## ✨ تایبەتمەندییەکان:
- **UI/UX مۆدێرن**: بەکارهێنانی ستایلی Glassmorphism.
- **Animations**: جوڵەی نەرم بە بەکارهێنانی GSAP.
- **Custom Cursor**: ئەزموونێکی کارلێکەری ناوازە بۆ بەکارهێنەر.
- **Fully Responsive**: گونجاوە بۆ هەموو شاشەیەک.

## 📂 پێکهاتەی فایلەکان:
1. `index.html`: لاپەڕەی سەرەکی.
2. `style.css`: هەموو دیزاین و ستایلەکان.
3. `script.js`: ئەنیمەیشن و کارلێکەکان.
4. `/templates`: فۆڵدەری قالبە ئامادەکراوەکان.
